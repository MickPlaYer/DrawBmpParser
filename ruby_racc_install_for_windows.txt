1. 安裝Ruby
	至http://rubyinstaller.org/downloads/下載Ruby 2.2.3安裝檔。
	安裝時可勾選加入環境PATH。
2. DEVELOPMENT KIT(https://github.com/oneclick/rubyinstaller/wiki/Development-Kit)
	至http://rubyinstaller.org/downloads/下載DevKit壓縮包。
	解壓縮至"C:/RubyDevKit/"(或其他常駐資料夾)。
	使用cmd移至剛剛的資料夾
	執行"ruby dk.rb init"
	以及"ruby dk.rb nstall"
	來完成DevKit的綁定。
3. Racc(https://github.com/tenderlove/racc)
	使用cmd執行"gem install racc"。
	完成後執行"racc"測試。
4.使用Racc
	產生Parser
	cmd>> racc -o "MyParser.rb" "racc.y"
	執行Ruby檔
	cmd>> ruby MyParser.rb
